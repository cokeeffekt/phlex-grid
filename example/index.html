Responsive

Responsive modifiers enable specifying different column sizes, offsets, alignment and distribution at small, medium, & large viewport widths.

<div class="row">
    <div class="small-12
                medium-8
                large-6
                ">
        <div class="box">Responsive</div>
    </div>
</div>

Collapse

Collapse Gutters

<div class="row collapse">
    <div class="small-12
                medium-8
                large-6
                col-lg-4">
        <div class="box">Responsive</div>
    </div>
</div>

Fluid

Percent based widths allow fluid resizing of columns and rows.

.small-6 {
  flex-basis: 50%;
}

Simple Syntax

All you need to remember is row, column, content.

<div class="row">
    <div class="small-12">
        <div class="box">12</div>
    </div>
</div>

Offsets

Offset a column

<div class="row">
    <div class="small-offset-3 small-9">
        <div class="box">offset</div>
    </div>
</div>

Auto Width

Add any number of auto sizing columns to a row. Let the grid figure it out.

<div class="row">
    <div class="small">
        <div class="box">auto</div>
    </div>
</div>

Nested Grids

Nest grids inside grids inside grids.

<div class="row">
    <div class="small">
        <div class="box">auto</div>
            <div class="row">
                <div class="small">
                    <div class="box">auto</div>
                </div>
            </div>
        </div>
    </div>
</div>

Alignment

Add classes to align elements to the start or end of a row as well as the top, bottom, or center of a column

.start-*

*default alignment
<div class="row small-start">
    <div class="small-6">
        <div class="box">
            start
        </div>
    </div>
</div>

.*-center

<div class="row small-center">
    <div class="small-6">
        <div class="box">
            start
        </div>
    </div>
</div>

.*-end

<div class="row small-end">
    <div class="small-6">
        <div class="box">
            end
        </div>
    </div>
</div>

Here is an example of using the modifiers in conjunction to acheive different alignment at different viewport sizes.
